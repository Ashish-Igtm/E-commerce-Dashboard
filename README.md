# E-commerce-Dashboard
This project focuses on analyzing ecommerce data to uncover the gap between business growth and customer satisfaction. The goal was not just to visualize data, but to identify hidden patterns that impact overall performance.

🚀 Project Overview

At first glance, the business shows strong growth — with high revenue, consistent orders, and a solid YoY increase. However, deeper analysis reveals that:

High revenue does not always mean high customer satisfaction
Delivery performance directly impacts customer ratings
Certain regions and categories show hidden inefficiencies

This project highlights how data can be used to move from surface-level metrics to actionable insights.

🛠️ Tools & Technologies
Power BI → Data visualization & dashboard creation
PostgreSQL (SQL) → Data extraction, transformation, and querying
💻 Data Preparation using PostgreSQL

In this project, I used PostgreSQL to handle raw data before visualization:

Imported raw data using CSV files
Created proper table structures and relationships (connections)
Performed data cleaning and transformation using SQL queries
Built custom views to simplify complex queries and improve analysis
Extracted aggregated data for Power BI dashboards

📈 Key Insights
📊 Revenue reached 14.21M with significant growth
⭐ Average rating ~ 4.03, but varies across categories
🚚 Late deliveries negatively impact customer experience
🌍 Some regions perform well in sales but poorly in delivery metrics
💡 High-performing categories still have scope for improvement in satisfaction

📂 Dashboard Features
Interactive KPI cards (Revenue, Orders, AOV, Ratings)
Category-wise and region-wise analysis
Delivery performance tracking (on-time vs late orders)
Customer rating distribution
📌 Drill-through feature for deep-level analysis (category → detailed view)

🎯 What I Learned
Data is not just about numbers — it’s about asking the right questions
Visualization becomes powerful only when backed by strong data logic (SQL)
Real insights come from going beyond dashboards and exploring deeper layers
Hands-on experience with PostgreSQL for real-world data handling

📌 Conclusion

This project demonstrates how combining PostgreSQL + Power BI can transform raw data into meaningful insights and support better decision-making.
